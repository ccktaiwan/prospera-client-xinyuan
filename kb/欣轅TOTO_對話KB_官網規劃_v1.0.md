## 二、品牌雙層架構（核心策略）

```
欣轅（企業總部品牌）
├── 官方名稱、願景、定位
├── 事業體層級架構
├── 專業敘事（工程 × 設計師合作 × 服務標準）
├── 全案例庫（含巍傑）
├── 企業合作（建案、設計師）
└── 聯絡 × 導覽 × 企業簡介

巍傑（技術品牌）
├── 工程專業與安裝能力
├── 施工案例（外溢來源）
├── 團隊介紹
├── 施工流程 × 風險控管
└── 定位為欣轅「工程事業部」
```

**品牌外溢邏輯：**
- 巍傑案例 → 上傳 → 欣轅整合成更高級的品牌故事
- 設計師看到的是欣轅 → 內部工程由巍傑執行
- 專業力向上集中（欣轅） / 產能向下執行（巍傑）
- 欣轅 = 品牌承接者，巍傑 = 專業外溢者

## 三、品牌傳播飛輪（已確認）

```
FB 觸發
→ 品牌 × 案例（巍傑提供內容）
→ 欣轅官網（品牌母艦 × 線索上包）
→ FB 回流（情緒 × 認知提升）
→ GSC（搜尋信任 × 權重累積）
→ LINE@（成交 × 設計師養成 × 經營）
→ 再回 FB → 飛輪持續轉動
```

**飛輪六大節點說明：**

| 節點 | 角色 | 核心任務 |
|------|------|---------|
| FB | 內容啟動 | 以巍傑案例建立第一觸點，重點是存在感而非互動 |
| 品牌 × 案例 | 內容源頭 | 安裝案例、工程前後對照、專業流程拆解 |
| 欣轅官網 | 品牌總部 | 吸納所有內容，形成品牌唯一可信來源 |
| FB 回流 | 情緒認知 | 官網專業內容反向燃料，強化熟悉度 |
| GSC | 搜尋信任 | Google 索引累積，長期不花錢的信任資產 |
| LINE@ | 成交養成 | 設計師合作、工程詢價、活動推播、長期維繫 |

### 工程案例庫 Projects ⭐（最重要）

> 本區是整個網站最重要的權威來源，也是巍傑案例外溢的主要管道

**案例分類（三層）：**

依施工類型：
- 整體改造
- 排水修繕
- 浴缸 / 花灑安裝
- 馬桶更新
- 防水工程

依空間條件：
- 小坪數
- 老屋翻新
- 新成屋調整
- 公共空間

依問題解決：
- 漏水
- 排水慢
- 異味
- 設備老化
- 動線不良

**單一案例頁模板（標準化，可複製）：**
```
① Before / After（雙欄對照圖）
② 案件背景（屋況、坪數、屋齡）
③ 問題診斷（顧客原始困擾）
④ 解決方案（欣轅 × 巍傑工程思路）
⑤ 使用設備（TOTO 型號 × 規格）
⑥ 工程流程（分步驟說明）
⑦ 完工照（多角度）
⑧ 設計師或顧客回饋（可後補）
⑨ 延伸閱讀（導回服務頁）
⑩ 立即諮詢 CTA（LINE@）
