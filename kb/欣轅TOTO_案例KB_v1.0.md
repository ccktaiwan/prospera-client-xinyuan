## 三、策略主張（Phase 架構）

**核心商業模型：B + C 雙軌**

| 軌道 | 現在 | 未來 |
|------|------|------|
| B 軌 | 品牌與系統顧問 | 持續深化 |
| C 軌 | — | 平台分潤 |

**三階段執行架構：**

```
Phase 1（現在執行）
目標：品牌權威建立
做：H1 Landing Pages / 工程案例 / 知識庫 / TOTO 深度內容
不做：電商 / AI 功能 / 選材庫

Phase 2（下一階段）
目標：會員工具層
做：設計師專區 / 選材系統 / 搜尋 / AI 初步導入

Phase 3（平台化）
目標：生態系建立
做：導購 / 分潤 / 3D 體驗 / AI 決策輔助
```

## 四、網站結構（Phase 1 定案）

**H1 五大入口頁：**

```
1. 首頁（品牌）
2. 服務項目
3. 工程案例
4. 知識庫
5. 設計師合作
```

**三層內容系統：**

```
品牌層：情緒 / 信任建立（設計雜誌語境）
工程層：專業 / 解法呈現
知識層：教育 / SEO 佈局
```

---

## 五、內容生產流程（標準化）

每一頁走以下六步：

```
Step 1：定義 H1（確認頁面主題）
Step 2：寫敘事（品牌語氣，設計雜誌語境）
Step 3：建立 H2/H3 結構（SEO + AI 爬蟲）
Step 4：生成 HTML（可立即上線）
Step 5：生成視覺示意圖（對齊設計師）
Step 6：交付工程上線（Wix / WordPress）
```
