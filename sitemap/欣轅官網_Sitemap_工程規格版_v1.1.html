重點頁面功能規格

P-04 工程案例庫

/projects & /projects/[slug]

列表頁功能

  • 格狀排列（3 或 4 欄）
  • 篩選器（Tag 多選）
  • 分頁或無限捲動
  • 縮圖 + 標題 + 分類

內頁區塊

  • Hero 大圖或輪播
  • Before / After 雙欄
  • 案件背景文字
  • 設備清單（型號）
  • 工程流程步驟
  • 完工多圖展示
  • 延伸閱讀（3 張 Card）
  • LINE@ CTA

後台 CMS 需求

  • 案例新增 / 編輯 / 刪除
  • 多圖上傳
  • Before / After 指定
  • Tag 管理
  • SEO meta 欄位
  • 發布 / 草稿狀態
⚠️ 案例內容由業主方提供，工程方只需建立後台上傳機制，不負責內容撰寫。

P-05 知識庫

/knowledge & /knowledge/[slug]

列表頁功能

  • 文章列表（含分類 Tab）
  • 分類：工程百科 / 選購指南 / 安裝科普
  • 縮圖 + 標題 + 摘要 + 分類
  • 分頁功能

內頁區塊

  • 文章標題
  • Rich text 內文（含圖）
  • 延伸閱讀自動推薦
  • LINE@ CTA
  • 麵包屑導覽

SEO 必要設定

  • 每篇 meta title
  • 每篇 meta description
  • OG image 指定
  • Canonical URL
  • Sitemap.xml 自動生成
  • 結構化資料（Article schema）
⚠️ 文章內容由業主方提供，工程方只需建立後台編輯機制與 SEO 欄位設定。
技術需求

平台建議

  • Wix / WordPress（業主確認中）
  • 需支援自訂 URL slug
  • 需支援 CMS 後台（案例 + 文章）
  • 需支援表單 + 檔案上傳
  • 行動裝置完全響應式

SEO 基礎設定

  • 每頁獨立 meta title / description
  • OG 標籤（Facebook / LINE 分享預覽）
  • Sitemap.xml 自動生成
  • Google Search Console 串接
  • Google Analytics 安裝

表單功能

  • 聯絡表單（含圖片上傳）
  • 合作申請表單（含圖面上傳）
  • 表單送出 → Email 通知
  • LINE@ 按鈕（全站 floating 可選）

圖片 / 媒體

  • 圖片壓縮與最佳化（WebP）
  • 多圖輪播元件
  • Before / After 比較元件
  • Lazy loading 必要
  • CDN 建議
