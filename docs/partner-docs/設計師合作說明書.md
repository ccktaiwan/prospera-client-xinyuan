# 欣轅 × 設計師合作說明書

> 適用對象：室內設計師、空間策劃者、軟裝整合者
> 版本：v1.0｜2026

## 一、為什麼跟欣轅合作？

欣轅是衛浴產業的工程整合服務商，獨特定位是把不可見的工程風險寫成可信任的標準。對設計師而言，這代表：

- **工程方說同一種語言**：欣轅交付的不是「報價單」，而是「工程說明書」，每一項材料、工法、時程都有對應依據
- **責任邊界清楚**：拆除、結構、防水、設備、收尾五段工序，每段都有獨立驗收節點
- **報價不藏匿**：所有設備型號、磁磚規格、五金品牌都列明，避免事後追加

## 二、三種合作方式

### 方式 A｜整案承包

設計師提供平面圖與設備清單，欣轅承包全室衛浴工程。

- 適合：設計師主導全案、但缺工程方
- 欣轅承擔：拆除、結構、管線、防水、設備、收尾
- 設計師承擔：客戶溝通、視覺決策、整體統籌

### 方式 B｜工程協作

設計師團隊有自己的工班，欣轅僅承接衛浴段。

- 適合：設計師主案中只想把衛浴外包給專業方
- 欣轅承擔：衛浴段所有工序
- 設計師承擔：其他空間 + 整體工序協調

### 方式 C｜選材顧問

設計師需要 TOTO 設備選型建議或工法諮詢，欣轅以時數或專案制提供顧問。

- 適合：設計師主案無外包需求、但需要設備規格諮詢
- 欣轅提供：選型建議、規格比對、現場勘查
- 不含：實際施工

## 三、合作流程

```
Step 1｜首次接觸
透過 LINE@ 或 email 聯繫，說明屋況、預算、時程。

Step 2｜現場勘查
欣轅工程師到現場，完成五項基本勘查（樓板、管線、防水、給水、電力）。

Step 3｜報價與工程說明書
出具報價單 + 工程說明書（雙文件），列明每段工序的工法、材料、時程。

Step 4｜合約簽訂
合約含付款節點（拆除、防水、設備、完工四階段）。

Step 5｜施工
依工序進行，每段完工後雙方驗收簽認。

Step 6｜完工交付
24 小時試水 + 排水測試 + 屋主交付。

Step 7｜保固
工程保固一年、設備依原廠保固。
```

## 四、欣轅的承諾

- **不偷料**：所有材料採購憑證可供檢視
- **不延工**：工序排程透明，每日進度可同步
- **不藏問題**：施工中發現的隱性問題即時通報，由雙方共同決策
- **不轉包**：核心工序由欣轅自有團隊執行，不層層外包

## 五、設計師最常問的問題（FAQ）

**Q1：合作要綁約多久？**
A：單案合作即可，不強制長期合約。第一次合作後，雙方覺得舒服，再談後續案件節奏。

**Q2：報價可以拆給屋主看嗎？**
A：報價單含工程說明書，雙文件都可給屋主。透明是欣轅的合作前提之一。

**Q3：欣轅會跟我搶客戶嗎？**
A：不會。透過設計師導入的案件，欣轅不會主動聯繫屋主延伸其他工程。所有後續溝通仍由設計師主導。

**Q4：可以指定品牌嗎？**
A：可以。欣轅最熟悉的是 TOTO，但其他品牌（凱撒、和成、Toto 競品）也都可執行。會在報價單註明是否為熟悉品牌。

**Q5：可以拿欣轅的案例給我的客戶看嗎？**
A：欣轅授權設計師合作夥伴使用已上線的官網案例。未上線的內部案例需個別取得授權。

## 結語

設計師與工程方的合作，最常見的爭議來自「沒講清楚的灰色地帶」。欣轅的合作邏輯，就是把所有灰色地帶在合約前先寫清楚，讓設計師可以專注在設計，而不必擔心工程的隱性風險。
